# 价格通知书_需商折

## 流程概述

1. **读取数据** — 读取 `eps3_mass.xlsx`
2. **时间筛选** — 根据用户输入的时间区间，筛选 `审批通过时间` 在该区间内的记录
3. **去重** — 按 `价格通知书号` 去重（保留第一条）
4. **商折分类** — 根据 `通知书备注` 字段中的关键词，判定是否为需商折记录
5. **筛选** — 保留 `是否商折字段` == "是" 的记录
6. **输出** — 保留 `价格通知书`、`供应商名称`、`通知书备注`、`创建人`、`识别日期` 字段
7. **累计追加** — 将结果追加到 `价格通知书_需商折/价格通知书_需商折.xlsx`（累计去重）

## 商折分类规则

### 分类为"是"的条件

- 通知书备注包含 "返" 字（强正向信号）
- 通知书备注包含 "商" 且匹配以下正向关键词之一：

| 关键词 | 说明 |
|--------|------|
| 追回 | 追回商折 |
| 返还 | 商折返还 |
| 抵扣 | 货款抵扣 |
| 扣款 | 月份扣款 |
| 签订商折 | 已签商折函 |
| 追商折 | 追回商折金额 |
| 需要商折 | 需补商折 |
| 涉及商折 | 涉及商折 |
| 存在商折 | 存在商折 |
| 商折抵扣 | 商折抵扣 |
| 一次性商折 | 一次性商折 |
| 商折追回、商折方式、商折返还 | 商折相关 |
| 货款中抵扣、开票中商折抵扣、商折扣款 | 各类抵扣场景 |
| 月扣款、月货款、月份扣款 | 月度扣款 |
| 商折 | 通用匹配 |

### 分类为"否"的条件

- 空值
- 包含"商降"（商业降价，非商折）
- 包含"商折不存在"
- 匹配 `商折 x%` 的模式
- 不包含"商"或"返"字

## 输出字段

| 字段 | 说明 |
|------|------|
| 价格通知书 | 价格通知书号 |
| 供应商名称 | 供应商名称 |
| 通知书备注 | 原始备注内容 |
| 创建人 | 创建人 |
| 识别日期 | 运行当日日期（格式：2026/06/05） |

## 数据依赖

| 文件 | 说明 |
|------|------|
| `eps3_mass.xlsx` | 主数据源（含审批通过时间、通知书备注等字段） |

## 输出

- `价格通知书_需商折/价格通知书_需商折.xlsx`（累计追加，按价格通知书号去重）
