# 94项需修改为联动标识检查

## 执行方式

在项目根目录下，用 VSCode 打开 notebook 并逐 cell 运行：

```
94项需改为联动标识检查.ipynb
```

或直接批量执行：

```bash
jupyter nbconvert --to notebook --execute --inplace 94项需改为联动标识检查.ipynb
```

## 数据依赖

运行前确保以下文件存在于项目目录：

| 文件 | 说明 |
|------|------|
| `备件_不联动清单_整改及跟踪项.xlsx` | 备件不联动清单（含对策列） |
| `eps3_spare.xlsx` | EPS备件输机价格数据（含通知书状态列） |

## 分析步骤概览

1. **Step 1** — 读取备件清单，筛选 `对策 == "不联动→联动（已确定）"`，构造 key1 = `备件号_供应商编码`
2. **Step 2** — 读取 eps3_spare，筛选 `通知书状态 == "已审批"`，构造 key2 = `备件号_供应商编码`
3. **Step 3** — 按 key2 分组：若生效/失效日期跨年，先将失效日期调至生效年最后一天；按失效日期降序 → 创建时间降序，取每组第一条
4. **Step 4** — key1 左连接 key2，将 key2 的 `价格通知书号`、`备件裸价`、`备件物流费`、`备件包装费`、`是否联动(0是，1否)` 字段添加到 key1 对应记录中，导出 Excel 结果

## 输出

- 终端打印 key1 数量 / key2 已审批数量 / 分组后数量 / 匹配统计
- 导出 `./联动标识检查结果/94项需改为联动标识检查结果_<日期>.xlsx`
