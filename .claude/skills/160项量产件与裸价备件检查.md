# 160项量产件与裸价备件检查

## 执行方式

在项目根目录下，用 VSCode 打开 notebook 并逐 cell 运行：

```
notebooks/160项量产件检查跟踪_mass.ipynb
```

或直接批量执行：

```bash
jupyter nbconvert --to notebook --execute --inplace notebooks/160项量产件检查跟踪_mass.ipynb
```

## 数据依赖

运行前确保以下文件存在于项目目录：

| 文件 | 说明 |
|------|------|
| `data/备件_不联动清单_整改及跟踪项.xlsx` | 备件不联动清单（含对策列） |
| `data/eps3_mass.xlsx` | EPS量产件输机价格数据（含状态列） |

## 分析步骤概览

1. **Step 1** — 读取备件清单，筛选 `对策 == "不联动→联动（推进中）"`，构造 key1 = `量产件号_供应商编码`
2. **Step 2** — 读取 eps3_mass，筛选 `状态 in ["已发布", "已确认", "已审批"]`，构造 key2 = `零件号_供应商编码`
3. **Step 3** — 按 key2 分组：若生效/失效日期跨年，先将失效日期调至生效年最后一天；按失效日期降序 → 创建时间降序，取每组第一条
4. **Step 4** — key1 左连接 key2，比对各行的备件裸价与零件裸价，新增列"备件裸价与最新输机量产件对比"（大于/等于/小于/无匹配），导出 Excel 结果

## 输出

- 终端打印匹配统计 + 价格对比分布
- 导出 `output/联动标识检查结果/160项持续跟踪检查结果_<日期>.xlsx`
